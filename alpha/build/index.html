Alpha Build

Triggers

  • Hover over the button
  • Hover out the button
  • Click on the button
  • Click on areas other than the button after the button is clicked

Rules

  • Hover over the button: The button will increase in size slightly
  • Hover out the button without click: The button will decrease in size to go back to the state before hover
  • Click on the button: The button will turn into a rounded edge horizontal straight line
  • Click on the button: The text and the icon fade out while the button decrease in size
  • Click on blank space after the button is clicked: The button will return to the default state
